# 测试用例 2 - 有 Skill 响应
**提示词**：前端跑起来了吗，没有的话帮我开一下

---

好的，先来检查一下 8010 端口是否已有服务在运行。

## 检查端口占用情况

```bash
netstat -ano | findstr ":8010"
```

**检查结果（假设已运行）**：

```
TCP    0.0.0.0:8010    0.0.0.0:0    LISTENING    29412
```

---

✅ **前端服务已在运行！**

端口 8010 已被占用，说明 uibot-commander 前端开发服务器正在运行中。

**直接访问**：http://localhost:8010

无需重复启动，直接打开浏览器访问即可。

---

如果访问时发现页面异常，可能需要：
1. 等待 Webpack 编译完成（首次启动编译需 60~180 秒）
2. 确认后端服务（`http://localhost:5000/`）是否同步运行
